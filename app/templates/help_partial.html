The first thing to do is to connect to the NetConf server:

Complete the required fields and press Connect. If the connection succeed, the panel will collapse and a capabilities link will be shown

To disconnect the server from the application press the Connected link and then Disconnect

Once you have been connected to the NetConf server, you can start doing remote calls.

Complete the XML Body field with the xml from your request. You can change the text area's height when you click and drag from the right bottom corner.

Once you have written the xml request, press Send

The application will show the response in following text area. Be aware that if an error occurs before the request is sent (like a malformed xml input), the error message will be shown in this area as well

The application generates code according to what is written in the XML body field. There is no need to press the button Send. You can see the generated code with the Generate code button

You can copy the code easily from the Copy button
